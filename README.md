# CertisTech Website (Next.js + Cloudflare Pages)

Dark premium, bilingual (AR default + EN auto-detect), enterprise aesthetic.

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run start
```

## Routing
- `/` redirects to `/ar` (default) or `/en` based on browser + saved cookie.
- Manual: `/ar`, `/en`

## Assets
- Hero background: `public/img/hero-bg.jpg` (replace with your final Banana output anytime)
- Logo placeholder: `public/logo/certistech.svg`
